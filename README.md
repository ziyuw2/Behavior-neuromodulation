### +audiHelper
Contains functions used to generate tones of different frequency

### +niHelper
Contains functions used to extract TTL from NI DAQ analog recording

### +task
Contains task script for auditory classical conditioning and Go/Nogo with a movable port.

### +taskHelper
Contains functions used in the task scripts

### +waterCalibration
Contains the water calibration data for the solenoids on different rigs

condductive_sensor_ziyu: uploaded to NANO for conductive lick detection
daq_ardu_directOutput: uploaded to uno for using UNO R3 as a DAQ
